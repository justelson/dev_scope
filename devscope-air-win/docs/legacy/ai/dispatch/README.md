# Assistant Dispatch Pack

This folder contains copy/paste-ready dispatch assets.

## Files

1. `backend_agent_prompt.txt`
2. `frontend_agent_prompt.txt`
3. `final_reviewer_prompt.txt`
4. `assistant_backend_log.csv`
5. `assistant_frontend_log.csv`
6. `assistant_reviewer_decisions.csv`

## Usage

1. Send backend prompt to backend agent.
2. Send frontend prompt to frontend agent.
3. Collect their reports.
4. Review using handoff checklist.
5. Record final decision row in `assistant_reviewer_decisions.csv`.
